# Manual figures (Milo 2.0)

Illustrations for the web manual will live beside the chapters that reference them, for example:

```
manual/img/z_axis_frame/step_01.png
```

Export matching images from the official PDF (or future CAD renders) and reference them with Markdown, for example:

```markdown
![Column overview](img/z_axis_frame/step_01.png)
```

Until those files exist, chapters include a note pointing builders to the PDF release in the [Milo-V2.0](https://github.com/MillenniumMachines/Milo-V2.0) repository.
