## Hardware glossary (from manual)

| Item | Description |
| --- | --- |
| BUTTON HEAD CAP SCREW (BHCS) | Low-profile metric fastener with a domed head and hex socket drive. |
| DROP IN NUT | Post-install fastener that slides into a T-slot to provide a threaded anchor point. |
| SOCKET HEAD CAP SCREW (SHCS) | High-strength metric fastener with a cylindrical head and hex socket drive. |
| BELLEVILLE WASHER | Conical spring washer providing high load in limited space. |
| FLAT HEAD CAP SCREW (FHCS) | Countersunk fastener with a flat top and hex socket drive for a flush finish. |
| HEAT INSERT | Threaded brass insert designed to be heated and embedded into plastic parts. |
| GREASE NIPPLE | Fitting used to apply lubricant into bearings or mechanical joints. |
| ROLLER BEARING | Rotating element that reduces friction between moving parts using steel balls. |
| NEODYMIUM MAGNET | High-strength rare-earth magnet offering powerful magnetic force in a compact size. |
| DOUBLE SPLIT SHAFT COLLAR | Clamp-style collar with two splits for secure, even clamping around a shaft. |
| THRUST WASHER | Flat bearing surface that supports axial loads and reduces friction. |
| SPIDER COUPLER | Flexible shaft coupling using an elastomeric spider to absorb misalignment and vibration. |
| FLAT WASHER | Flat disc used to distribute load and prevent surface damage under a fastener. |
| TIMING PULLEY | Toothed pulley designed to engage with a timing belt for precise, slip-free power transmission. |

#### Miley only

| Item | Description |
| --- | --- |
| SET SCREW | Headless threaded fastener used to secure one component to a shaft through side pressure. |
| DOWEL PIN | Precision cylindrical pin used for accurate alignment and positioning of components. |
| SHOULDER BOLT | Precision fastener with an unthreaded shoulder for guiding, pivoting, or locating parts. |
| LEADSCREW NUT | Threaded brass component that converts rotary motion to linear motion on a leadscrew. |
