# FMJ (V2)

The **Full Metal Jacket (FMJ) V2** option replaces selected printed Z-column parts with metal plates for extra rigidity. In the exported manual, **FMJ V2** and **FMJ PLATES** first appear during the **Z axis frame** build.

## Where to work

Complete the FMJ steps **in context** while you assemble the column:

- [Z Axis Frame](50_z_axis_frame.md) — locate the **FMJ V2** / **FMJ PLATES** sections and install those parts when the manual calls for them.

If you are not installing an FMJ kit, skip those substeps and continue with the printed alternatives shown in the same chapter.
