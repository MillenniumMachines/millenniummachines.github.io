# Milo V2 - Sourcing guide

This page is a **placeholder** while vendor listings and kit SKUs for Milo / Miley V2 are consolidated for the documentation site.

- **Do not assume** V1.5 sourcing tables apply directly: frame extrusions, plates, screws, and motion parts differ for V2.
- Watch the [Milo-V2.0](https://github.com/MillenniumMachines/Milo-V2.0) repository for published BOM tables, recommended suppliers, and community-sourced options.

When you are ready to source fasteners and motion hardware, use the CAD and released BOM from that repository together with the **[official assembly manual (PDF)](https://github.com/MillenniumMachines/Milo-V2.0/blob/main/Manual/MiloV2-Manual.pdf)**.
